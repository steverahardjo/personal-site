Application Scraper

Automated Job Application Tracking from Email

Overview

An automated system for extracting and summarizing job application data from email communications. The scraper processes inbox messages and organizes application information into a structured tabular format.

This tool helps job seekers track their applications across multiple platforms without manual data entry.

Features

  • Email Extraction: Parse job-related emails by sender and subject
  • Auto-Summarization: Extract key information using NLP
  • Structured Output: Generate organized tables with application data
  • Platform Detection: Identify application platforms (LinkedIn, Indeed, etc.)
  • Status Tracking: Track application progress automatically
  • Export Options: Output to CSV, JSON, or Google Sheets

Data Fields

The system extracts the following information from each application:

  • Unique ID: Auto-generated identifier
  • Sender: Company/recruiter information
  • Job Title: Position applied for
  • Platform: Application source
  • Process Stage: Current status in hiring pipeline
  • Chained Email ID: Thread tracking

Architecture

The system consists of two main agents:

  • Extractor Agent: Scans Gmail inbox for job-related messages
  • Summarizer Agent: Processes email content and extracts entities

Tech Stack
