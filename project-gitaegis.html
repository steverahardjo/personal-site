GitAegis

Secret Detection CLI Tool for Git Workflows

Overview

GitAegis is a lightweight Go CLI tool designed to help developers maintain clean and secure repositories by scanning source code for potential API keys, tokens, or sensitive data before committing.

It uses Shannon entropy analysis combined with regex pattern matching and tree-sitter AST parsing to detect secrets with high accuracy and minimal false positives.

Features

  • Secret Detection: Identify API keys, tokens, and sensitive patterns
  • Entropy Analysis: Shannon entropy calculation for random string detection
  • AST Parsing: Tree-sitter integration for language-aware scanning
  • Git Integration: Hook into git add or scan entire directories
  • Logging: JSON-formatted logs for CI/CD integration
  • High Performance: Goroutine concurrency for fast scanning

Installation

Bash 
curl -L "https://github.com/steverahardjo/gitaegis/releases/latest/download/gitaegis-linux-amd64" -o /tmp/gitaegis && \
chmod +x /tmp/gitaegis && \
sudo mv /tmp/gitaegis /usr/local/bin/gitaegis

Usage

Bash 
# Scan entire directory with logging
gitaegis scan . --logging

# Scan only git-staged files
gitaegis scan . -g

# Update .gitignore based on detected files
gitaegis gitignore

Technical Details

The tool implements several core components:

  • analyzer: Core scanning logic with goroutine-based parallel processing
  • entro_parser: Line-by-line parser with entropy threshold filtering
  • sitter: Go tree-sitter bindings for AST-based detection
  • file_modification: Obfuscation and gitignore management
